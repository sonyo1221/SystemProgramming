📌 프로젝트 소개

Comment Editor는 시스템 프로그래밍 수업에서 배운 파일 I/O, 버퍼링, 포맷팅, 문자열 처리, 동적 메모리 관리 등을 활용하여 제작한
주석 기반 텍스트 편집 도구입니다.

이 프로그램은 코드 리뷰가 필요한 환경을 가정하여,

개발자 모드(Developer Mode)

리뷰 모드(Review Mode)

복구 모드(Restore Mode)
를 지원하며, 실제 협업 환경에서 발생하는 주석 추가, 자동 주석 템플릿 생성, 리뷰 기록 관리, 리뷰 삽입, 백업 및 복구 기능을 제공합니다.

✨ 주요 기능 (Key Features)
🔧 1. Developer Mode (개발자 모드)

특정 라인에 주석 추가

함수 정의 자동 탐지 → 주석 템플릿 자동 삽입

TODO/FIXME/NOTE 자동 강조

오타 감지 기능 (Levenshtein Distance 기반 Suggestion)

코드 내 주석만 추출하여 별도 파일 생성

포맷팅된 파일 자동 생성

원본 파일 손상 방지를 위한 자동 백업 수행

📝 2. Review Mode (리뷰 모드)

코드 전체 보기, 구간 보기, 키워드 검색

라인 기반 리뷰 코멘트 작성

review_notes.txt에 파일별 리뷰 자동 저장

리뷰 코멘트를 실제 파일에 삽입하여 반영하는 기능

미리보기 형식으로 화면에만 리뷰 삽입하는 기능

리뷰 적용 시 원본 파일 자동 백업

🛟 3. Restore Mode (복구 모드)

백업된 .bak 파일을 감지하여 원본 파일로 복원

백업이 없을 경우 폴더의 .bak 목록 자동 표시

실수로 파일을 덮어쓴 경우 즉시 되돌릴 수 있음
